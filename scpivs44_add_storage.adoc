= 新增儲存設備
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
請依照本節中的步驟新增儲存設備。

. 在選擇控制閥外掛程式的左導覽器窗格中、按一下 * 儲存系統 * 、然後選取 * 新增 * 選項。
. 在 Add Storage System （新增儲存系統）對話方塊中、輸入基本 SVM 或叢集資訊、然後選取 *Add* （ * 新增 * ）。

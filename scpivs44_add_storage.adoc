= 新增儲存設備
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


. 在VMware vSphere Web用戶端的左側導覽器窗格中、按一下「*儲存系統*」、然後按一下 image:scpivs44_image6.png["錯誤：缺少圖形影像"] *新增*。
. 在「新增儲存系統」對話方塊中、輸入基本SVM或叢集資訊、然後按一下「*新增*」。

= 建立備份原則
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


. 在VMware vSphere Web用戶端的左側導覽器窗格中、按一下*原則*、然後按一下 image:scpivs44_image6.png["錯誤：缺少圖形影像"] *新政策*。
. 在「*新備份原則*」頁面上、輸入原則組態資訊、然後按一下「*新增*」。
+
如果原則將用於鏡射保存庫關係、則在「複寫」欄位中、SnapVault 如果您想要將備份複製到鏡射保存庫目的地、則只能選取「*更新備份後的更新*」選項。

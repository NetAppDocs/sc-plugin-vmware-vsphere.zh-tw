= 建立備份原則
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
請依照下列指示建立備份原則

. 在選擇控制閥外掛程式的左導覽器窗格中、按一下 * 原則 * 、然後選取 * 新原則 * 。
. 在「*新備份原則*」頁面上、輸入原則組態資訊、然後按一下「*新增*」。
